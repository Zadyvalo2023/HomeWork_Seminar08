	Задача 54: Задайте двумерный массив. Напишите программу, которая упорядочит по убыванию элементы каждой строки двумерного массива.
Например, задан массив:
1 4 7 2
5 9 2 3
8 4 2 4
В итоге получается вот такой массив:
7 4 2 1
9 5 3 2
8 4 4 2


===================================================================

Данный код  объявляет двумерный массив array и инициализирует его значениями. Затем вызывается функция SortRowsDescending, которая сортирует строки массива по убыванию. Для этого используется алгоритм сортировки пузырьком. После сортировки массив выводится на консоль с помощью функции PrintArray 
Функция SortRowsDescending принимает двумерный массив в качестве аргумента и сортирует каждую строку массива по убыванию. Для этого используется алгоритм сортировки пузырьком, который проходит по каждой строке массива и сравнивает пары соседних элементов. Если текущий элемент меньше следующего, то они меняются местами. Алгоритм продолжает проходить по строке до тех пор, пока все элементы не будут отсортированы по убыванию
Функция PrintArray принимает двумерный массив в качестве аргумента и выводит его значения на консоль. Для этого используется вложенный цикл for, который проходит по каждому элементу массива и выводит его значение на консоль. После вывода всех элементов строки происходит переход на новую строку
