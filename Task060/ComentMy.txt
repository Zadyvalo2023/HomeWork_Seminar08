Задача 60. ...Сформируйте трёхмерный массив из двузначных чисел. Напишите программу, которая будет построчно выводить массив, добавляя индексы каждого элемента.
Массив размером 2 x 2 x 2
66(0,0,0) 25(0,1,0)
34(1,0,0) 41(1,1,0)
27(0,0,1) 90(0,1,1)
26(1,0,1) 55(1,1,1)
====================================================================================

- Код начинается с использования директивы  using System; , которая позволяет программе использовать классы и методы из пространства имен  System . 
-  class Program  определяет класс с именем  Program . 
-  static void Main()  является точкой входа в программу. 
- Внутри метода  Main  создается трехмерный массив  array  размером 2 x 2 x 2 и заполняется двузначными числами. 
- Затем код выводит массив построчно с добавлением индексов каждого элемента. Внешний цикл перебирает первую размерность массива, средний цикл перебирает вторую размерность, а внутренний цикл перебирает третью размерность. Для каждого элемента массива выводится его значение и соответствующие индексы в формате  значение(индекс1,индекс2,индекс3) .
